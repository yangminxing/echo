# 固定表单设计器 FixedFormDev 开发计划

## 概述
基于 Vue3 + TypeScript + Vite 技术栈，在不使用第三方 UI 组件库的情况下，开发固定表单设计器页面。

## 实现步骤

### 1. 列表页面设计
**文件**: `FixedFormDev.vue`

#### 1.1 顶部过滤区域
- **过滤列**:
  - 表类型: 下拉选择框
  - 表描述: 文本输入框
- **按钮**: "建表" 按钮

#### 1.2 列表区域
使用原生 `<table>` 实现，包含以下列：
| 表类型 | 表名 | 表描述 | 版本 | 同步状态 | 创建时间 | 操作 |
- 操作列包含 "编辑" 按钮

#### 1.3 数据模型
```typescript
interface TableInfo {
  tableType: string;      // 表类型
  tableName: string;       // 表名
  tableDesc: string;       // 表描述
  version: number;         // 版本
  syncStatus: string;      // 同步状态
  createTime: string;      // 创建时间
}
```

### 2. 表单编辑弹窗设计
**组件内嵌**: 在 FixedFormDev.vue 中使用 `<dialog>` 或条件渲染实现

#### 2.1 基础信息区域
| 字段 | 控件类型 |
|------|----------|
| 表名 | 文本输入框 |
| 表描述 | 文本输入框 |
| 表类型 | 下拉选择框 |
| 表单分类 | 下拉选择框 |
| 主键策略 | 下拉选择框 |
| 显示复选框 | 开关/复选框 |
| 主题模板 | 下拉选择框 |
| 表单风格 | 下拉选择框 |
| 滚动条 | 开关/复选框 |
| 是否分页 | 开关/复选框 |
| 是否树 | 开关/复选框 |

#### 2.2 字段配置区域
- **新增按钮**: 添加字段
- **字段列表** (带复选框):
  | # | 字段名称 | 字段备注 | 字段长度 | 小数点 | 默认值 | 字段类型 | 主键 | 允许空值 | 同步数据库 |
- **删除按钮**: 选中字段后显示

#### 2.3 底部按钮
- 关闭按钮
- 保存按钮

### 3. 状态管理
使用 Vue3 `ref` 和 `reactive` 管理：
- 列表数据
- 当前编辑的表单数据
- 弹窗显示/隐藏状态
- 选中字段集合

### 4. 样式设计
- 使用 CSS Grid/Flexbox 布局
- 表格使用原生 `<table>` 标签
- 弹窗居中显示
- 统一按钮和输入框样式

## 文件结构
```
frontend/src/views/developer/FixedFormDev.vue
```

## 关键实现细节
1. 使用 `<script setup lang="ts">` 语法
2. 不引入任何第三方 UI 组件库
3. 所有组件使用原生 HTML 元素实现
4. 样式使用 scoped 避免污染
